scala.org.mx
============

Sitio y Blog colaborativo de la Comunidad Scala México

# Utilizamos Jekyll
Este es un blog que se basa en el proyecto de [Jekyll](http://jekyllrb.com/).

## Instalación y uso
    bundle install
    jekyll serve
